### GitHub User Finder

GitHub User Finder is a web-based application that allows users to seamlessly search for and retrieve profile information of any GitHub user. By leveraging the GitHub REST API, the application fetches real-time data and presents it in a clean, user-friendly interface.

The project is built entirely with vanilla web technologies, demonstrating a strong understanding of DOM manipulation, asynchronous JavaScript, and responsive design without the reliance on heavy frontend frameworks.

Key Features:

Real-Time Search: Users can input a GitHub username to instantly fetch and view profile details.

Profile Insights: Displays crucial user information including the avatar, bio, number of public repositories, followers, and following count.

Direct Integration: Provides quick links to the searched user's actual GitHub profile and repositories.

Responsive UI: Styled to provide a smooth and visually appealing experience across both desktop and mobile devices.

Tech Stack:

Frontend: HTML5, CSS3, JavaScript (Vanilla ES6+)

API: GitHub REST API (for fetching user data)

Deployment: Hosted live on Vercel
